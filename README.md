# vue-study

> A Vue.js project

## Build Setup

``` bash
#下载本项目

git clone https://github.com/wangyuantoy/vue-project-books.git

# 安装依赖
npm install

# serve with hot reload at localhost:8080

cd mock
node server.js  启动本地服务
npm run dev   启动项目
```

- mock 模拟数据和服务
- api 代表所有接口
- components 页面组件

# 主要功能

- 热门图书
- 删除图书
- 详情页跳转
- 修改图书
- 添加图书
- 实现loading效果
- 页面缓存
